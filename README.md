Описание проекта
Movie Library - это десктопное приложение для ведения личной коллекции фильмов. Программа позволяет добавлять, просматривать, фильтровать и удалять фильмы, а также сохраняет всю информацию в JSON-файл.

Основные возможности:
✅ Добавление фильмов - все поля с валидацией данных
✅ Просмотр коллекции - удобная таблица с информацией о фильмах
✅ Фильтрация - по жанру и году выпуска
✅ Удаление - удаление ненужных фильмов
✅ Статистика - общее количество фильмов и средний рейтинг
✅ Сохранение - автоматическое сохранение в JSON
✅ Визуализация - звёздочки для рейтинга, цветовая индикация

Валидация данных:
Год выпуска: целое число от 1888 до текущего года
Рейтинг: число от 0 до 10 (включительно)
Название: не пустое, не более 100 символов
Жанр: не пустой
